⚙️ Connect Your Google Sheet

One-time setup — no code changes needed after this.

  1. Open your Google Sheet in a browser.
  2. File → Share → Publish to web
    Set "Entire Document" + "Comma-separated values (.csv)" then click Publish.
  3. Copy the published CSV URL
    It looks like:
    https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/pub?output=csv
  4. Paste it below and click Save.
💡 Make sure you published as CSV, not HTML.
OR — Paste sample data to test the dashboard
